| n 范围 | 一般可接受复杂度   |
| ---- | ---------- |
| 10   | 阶乘         |
| 20   | O(2^n)     |
| 100  | O(n^3)     |
| 1000 | O(n^2)     |
| 1e5  | O(n\log n) |
| 1e6  | O(n)       |
| 1e7  | 卡常数才能勉强过   |

---

| 数据范围   | 应该想到      |
| ------ | --------- |
| `1e5`  | 不能双循环     |
| `1e6`  | 必须线性      |
| `2000` | 二重循环可能能过  |
| `100`  | 三重循环可能能过  |
| `20`   | 可以爆搜/状态压缩 |
