## yous

合并时不更新全体成员

查询时再慢慢更新

而且一个节点最多被压缩几次












```c++
int Find(int x)
{
    if(parent[x] == x)
        return x;
    return parent[x] = Find(parent[x]);
}
//连接根节点
void Union(int a, int b)
{
    if(Find(a) != Find(b))
    {
        parent[Find(a)] = Find(b);
    }
}

```
