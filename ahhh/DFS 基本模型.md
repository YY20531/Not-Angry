## 模型

| 模型        | 解决什么问题  | 核心思路           | 关键变量         | 是否用 used | 典型写法       |
| --------- | ------- | -------------- | ------------ | -------- | ---------- |
| 🔵 组合型DFS | 从n个数选k个 | “从哪里开始选”       | start        | ❌不用      | dfs(start) |
| 🟢 排列型DFS | n个数全排列  | “每个都可以选，但不能重复” | depth + used | ✔用       | dfs(depth) |
| 🟣 子集型DFS | 所有子集    | “选 or 不选”      | index        | ❌不用      | dfs(i)     |


## 作用

  | 类型        | 典型问题        | DFS在做什么    | 常用技巧                      | 关键词         |
| --------- | ----------- | ---------- | ------------------------- | ----------- |
| 🔢 组合枚举   | 选k个数、子集、全排列 | 枚举所有选择方案   | 回溯（push/pop）、used / start | “选谁”“怎么选”   |
| 🌲 树遍历    | 二叉树遍历、文件目录  | 访问节点 + 子节点 | 递归函数                      | “左右子树”      |
| 🧱 图搜索    | 连通性、路径问题    | 从一点走到所有可达点 | vis数组防重复                  | “能不能到”“连通块” |
| 🏝️ 网格搜索  | 岛屿数量、迷宫     | 四方向扩展搜索    | dx/dy数组                   | “上下左右走”     |
| 🎯 搜索所有方案 | 八皇后、数独      | 枚举+剪枝      | 限制条件判断                    | “合法方案”      |
| 🔁 状态搜索   | 括号生成、表达式    | 构造所有合法状态   | 剪枝                        | “构造所有可能”    |
